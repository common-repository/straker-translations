== Installation ==

This section describes how to install the plugin:

1. Upload the `straker-translations` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I have used Straker Translations before. Can I use the same account? =

Yes. During the registration step of the setup, simply provide the same email address that you have used in the past.
